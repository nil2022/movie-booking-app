# Movie Booking App

## Description
This is a Movie Booking app where customers can browse movies and book tickets of their choice.

## Installation

To set up the Movie Booking App locally, follow these steps:

1. Clone the repository to your local machine.
2. Run `npm install` to install all the dependencies.
3. Create a `.env` file in the root directory and set up your environment variables.
4. Run `npm start` to start the application.

## Usage

After starting the app, navigate to `http://localhost:3000` in your browser to start booking your movie tickets.

## Features

- Browse movies
- Book tickets
- User authentication
- View booking history

## Scripts

- `npm start`: Run the app in production mode.
- `npm run dev`: Run the app in development mode using nodemon for auto-restarting.

## Dependencies

- express
- mongoose
- dotenv
- bcrypt
- jsonwebtoken
- morgan
- cors
- validator
